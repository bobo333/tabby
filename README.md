# Tabby

### Usage

Run:
```
make run
```

Compile:
```
make compile
```

Clean:
```
make clean
```

Check with `Valgrind`:
```
make memcheck
```

General:
```
make compile
./tabby [path to mp3 to analyze]
```

---

Note: the example mp3 is a free sample, downloaded from [here](http://www.noiseaddicts.com/free-samples-mp3/?id=1449).
